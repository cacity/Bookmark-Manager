隐私政策 / Privacy Policy

我们非常重视用户隐私。本隐私政策旨在说明《书签管家》Chrome 插件如何处理您的信息。

We take your privacy seriously. This policy explains how the Bookmark Manager Chrome extension handles your information.

📌 本插件不会收集任何用户个人数据

📌 This extension does NOT collect any personal data

  • 本插件不会收集、存储或传输任何用户的个人信息或浏览记录。
  • 所有操作均在用户本地浏览器中完成，不涉及网络传输。
  • 插件不使用 Cookie、分析代码或任何形式的追踪技术。
  • This extension does not collect, store, or transmit any personal information or browsing data.
  • All processing is performed locally in your browser. No external servers are involved.
  • The extension does not use cookies, analytics, or any tracking technologies.

📋 使用的权限及说明 / Permissions and Usage

为了实现核心功能，插件使用了以下 Chrome 权限：

To enable its core functionality, this extension uses the following Chrome permissions:

  • bookmarks - 用于读取与管理书签数据 / Read and organize your bookmarks
  • storage - 用于保存插件设置 / Store plugin settings locally
  • downloads - 用于导出书签为 HTML 文件 / Export bookmarks to an HTML file
  • alarms - 定时同步和自动分类 / Schedule auto-classification and updates
  • sidePanel - 在侧边栏中显示书签界面 / Display bookmarks in a side panel UI

上述权限仅用于插件自身功能实现，所有数据处理仅限本地。

These permissions are used strictly for functionality, with all data processed locally in your browser.

🔐 数据安全 / Data Security

插件不会与任何第三方服务器进行数据通信，也不会将任何数据分享、出售或传输给第三方。

This extension does not communicate with any external server. No data is shared, sold, or transmitted to third parties.

📅 政策更新 / Policy Updates

如果我们对隐私政策进行更新，我们将在此页面上发布变更。

If we make changes to this privacy policy, we will update them on this page.
